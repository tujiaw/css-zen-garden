CSS Zen Garden

The Code

This page is designed to help you participate in the Zen Garden. Below are two essential files you'll need to get started:

To submit a design to the CSS Zen Garden, you'll need to download two files and modify only the CSS file. The HTML file must remain unchanged to ensure all submissions can be compared properly.

Important: Do not modify the HTML structure. Only change the CSS to create your design!

Download the Files

1. HTML File

This is the base HTML structure. Save it as index.html and do not modify it.

↓ Download index.html

2. CSS File

This is your starting point for CSS. Modify this file to create your design.

↓ Download style.css

Submission Guidelines

  • Keep the HTML structure exactly as provided
  • All visual changes must be done through CSS
  • Use valid CSS and test in multiple browsers
  • Submit your design via GitHub: github.com/mezzoblue/csszengarden.com
