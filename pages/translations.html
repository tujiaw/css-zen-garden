About the Translations

The CSS Zen Garden has been translated into multiple languages by volunteers around the world. These translations help make the project accessible to a global audience.

Available Translations

Due to the recent migration to a Node.js-based architecture, we are currently working on re-establishing the translation system.

If you are interested in helping with translations or would like to contribute a new language, please visit our GitHub repository:

github.com/mezzoblue/csszengarden.com

Historical Translations

Over the years, the CSS Zen Garden has been translated into numerous languages including:

  • Chinese (Simplified & Traditional)
  • Dutch
  • French
  • German
  • Italian
  • Japanese
  • Korean
  • Portuguese
  • Russian
  • Spanish
  • And many more...
